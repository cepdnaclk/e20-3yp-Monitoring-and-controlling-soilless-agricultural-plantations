Hardware Modules

Modular Design for Maximum Flexibility and Scalability

Environmental Sensor Module

Monitors critical environmental conditions for optimal plant growth

Key Measurements:
  • Temperature: Ambient air temperature monitoring
  • Humidity: Relative humidity levels
  • Light Intensity: Photosynthetically active radiation
Components:
  • DHT22 Temperature & Humidity Sensor
  • GY-30 Light Intensity Sensor
  • ESP32 Microcontroller
  • Battery Management System (BMS)
  • Weatherproof Enclosure

Water Sensor Module

Comprehensive water quality and level monitoring system

Key Measurements:
  • Water Temperature: Nutrient solution temperature
  • pH Level: Acidity/alkalinity monitoring
  • Water Level: Reservoir level detection
  • EC (Conductivity): Nutrient concentration
Components:
  • pH4502C pH Sensor
  • MD0591 Turbidity Sensor
  • PSR-22-B2 Water Level Sensor
  • ESP32 Microcontroller
  • Battery Management System (BMS)

Controller Module

Automated control system for nutrient delivery and water management

Key Functions:
  • Fertilizer Dispensing: Precise nutrient delivery
  • pH Control: Automatic pH adjustment
  • Water Management: Excess water disposal
  • Auto Refill: Low water level response
Components:
  • Fertilizer Dispensing System
  • Water Pumps (3 units)
  • Control Valves & Actuators
  • ESP32 Control Unit
  • Battery Management System (BMS)

System Components

User Interface

Cloud Platform

Database and API server for data storage, processing, and real-time communication

IoT Communication

Wireless communication module enabling real-time data transmission and control signals

System Data Flow

1
Sensor Data Collection

Environmental and medium sensors gather real-time data

2
Data Transmission

IoT module sends data to cloud platform

3
Cloud Processing

Database stores data, API processes requests

4
Mobile Interface

User monitors system status and sends commands

5
Control Signals

Commands sent to controller and fertilizer dispenser

6
Automated Response

System adjusts nutrients and environmental conditions
